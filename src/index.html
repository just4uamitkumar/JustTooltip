Add style.css to your head section

<link rel="stylesheet" type="text/css" href="style.css"/>

Use JQuery library. Minimum version shoule be of 1.4.0.

<script src="jquery-3.2.1.min.js"></script>

Add Tooltip jquery library

<script src="jquery.tooltip.js"></script>

Add below method to your custom code

$(document).ready(function(){
    $('a[data-title]').tooltip();
});

Add data-title with your anchor tag and content which you want to show on tooltip.

<a href='#' data-title='Your content'></a>

Choose your tooltip direction by using data-placement with your anchor tag. You can use "left", "right", "top" and "bottom" for tooltip direction. top will be default direction.

<a href='#' data-placement='right'></a>
